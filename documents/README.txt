Dossier pour les documents téléchargeables du site IAassiste.

À placer ici :
- certificat-qualiopi.pdf : le certificat Qualiopi (PDF) délivré par l'organisme certificateur.
  Le lien "Certificat Qualiopi" sur le site pointe vers ce fichier.

Badge Qualiopi (affiché en pied de page et sur Contact) :
- Le site utilise images/badge-qualiopi.svg (badge de substitution). Vous pouvez le remplacer par le logo officiel fourni dans le kit de votre organisme certificateur (en respectant la charte d'usage Qualiopi).
